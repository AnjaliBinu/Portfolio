# 🌐 Personal Interactive Portfolio

Welcome to my personal portfolio website repository! This is a single-page, responsive dashboard designed to showcase my academic background, technical skills, open-source contributions and engineering projects in a clean, interactive grid layout.

## 🔗 Live Demo
Check out the live website here: **https://anjalibinu.github.io/Portfolio/**

## ✨ Features
- **Interactive Card Layout:** Clicking on any grid card dynamically toggles an overlay pop-up window containing detailed insights.
- **Dynamic SVG Animation:** Features a custom vector flower that organically blossoms on a timed CSS keyframe loop, elegantly anchored in the bottom-left corner.
- **Fully Responsive Design:** Adapts smoothly across wide desktop layouts down to vertical single-column mobile screens.
- **Modern Typography & Styling:** Built using Google Fonts (Poppins/Inter) and crisp FontAwesome vector icons.

## 🛠️ Tech Stack & Tools Used
- **HTML5:** Semantic structural markup.
- **CSS3:** Flexbox, CSS Grid layouts, custom interactive keyframe animations, and overlay responsive configurations.
- **JavaScript (ES6):** Vanilla DOM manipulation for event listener attachments and modal display toggling.
- **Figma:** UI/UX design tool used to prototype, sketch, and wireframe the visual layout and card structures before coding.

## 📂 Project Structure
```text
├── index.html       # Main structural layout and modal overlay data templates
├── style.css        # Core design properties, grid configurations, and flower keyframes
├── script.js        # Event handling logic to toggle modal active visibility states
└── README.md        # Project overview and documentation
```
